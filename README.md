# 🌸 Data Classification Using AI

## 📌 Project Overview

This project demonstrates how to perform data classification using Artificial Intelligence and Machine Learning techniques. The model is trained on the **Iris Dataset (`irisdataset.csv`)** to classify flowers into different species based on their measurements.

The project includes data preprocessing, visualization, model training, prediction, and performance evaluation using Python in Jupyter Notebook.

## 🎯 Objectives

* Load and explore the Iris dataset.
* Preprocess the data for machine learning.
* Visualize data using graphs.
* Train a classification model.
* Predict flower species from input data.
* Evaluate model accuracy and performance.

## 🛠️ Technologies Used

* Python
* Jupyter Notebook
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn

## 📂 Dataset

* **Dataset Name:** `irisdataset.csv`
* Features:

  * Sepal Length
  * Sepal Width
  * Petal Length
  * Petal Width
* Target:

  * Iris-setosa
  * Iris-versicolor
  * Iris-virginica

## ⚙️ Project Workflow

1. Import required libraries.
2. Load the dataset.
3. Explore and clean the data.
4. Visualize relationships between features.
5. Split the dataset into training and testing sets.
6. Train a machine learning classification model.
7. Make predictions.
8. Evaluate the model using accuracy and other performance metrics.

## 📊 Results

The trained model successfully classifies Iris flower species with high accuracy. The project demonstrates the complete machine learning classification workflow from data preprocessing to prediction and evaluation.



## 🚀 How to Run

1. Clone the repository.
2. Open the project in Jupyter Notebook.
3. Install the required libraries:

   ```bash
   pip install pandas numpy matplotlib seaborn scikit-learn
   ```
4. Run the notebook cells in order.
